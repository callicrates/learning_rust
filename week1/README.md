# Notes on Week 1 Lab 2

I used Copilot to write the Fibonacci implementations in main.rs.

The instructions for this lab are for me to list something I like about Copilot pair programming.  I suppose it's OK when you already know what you want to write and how to do it.  I have big concerns relying on it when I'm just starting to learn a language, though.  There is value in doing the work by hand, thinking through whether to loop by index or over a range of elements, how to return a value... all the basic mechanics of a given language.  I will generate code more quickly but learn far more slowly if a tool does it for me.

Also, I am not a student and have not been for a long time, so I've already used probably 10 of my 50 allowed Copilot requests for the month just on this assignment.
